# HTTPX Tenacity

A lightweight package that provides tenacious (retrying) httpx transports.

## Installation

The package is published on [PyPI](https://pypi.org/project/httpx-tenacity/). Install it,
for example, with pip.

```sh
pip install httpx-tenacity
```
